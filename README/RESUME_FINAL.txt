🚀 ÉTAPES RAPIDES POUR COMMENCER
═══════════════════════════════════════════════════════════════════

1. Import base de données :
   phpMyAdmin → Importer database.sql

2. Configurer .env :
   DB_DATABASE=cauris_db

3. Lancer migrations :
   cd /opt/lampp/htdocs/backendCauris
   php artisan migrate

4. Installer et démarrer WebSocket :
   cd websocket-server
   npm install
   npm start

5. Tester l'API :
   curl http://localhost/backendCauris/public/api/health

6. Tester WebSocket :
   Le serveur écoute sur http://localhost:3000
