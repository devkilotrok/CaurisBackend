🚀 DÉMARRAGE
═══════════════════════════════════════════════════════════════════════

1. Installer Node.js dependencies :
   cd /opt/lampp/htdocs/backendCauris/websocket-server
   npm install

2. Démarrer le serveur WebSocket :
   npm start

3. Le serveur écoute sur http://localhost:3000

📡 ÉVÉNEMENTS DISPONIBLES
═══════════════════════════════════════════════════════════════════════

┌───────────────────────────────────────────────────────────────────┐
│ CLIENT → SERVEUR (Flutter émet)                                  │
└───────────────────────────────────────────────────────────────────┘

• join_room       → Rejoindre une salle
• leave_room      → Quitter une salle
• play_card       → Jouer une carte
• make_announcement → Faire une annonce
• start_game      → Démarrer le jeu
• trick_won       → Annoncer un pli gagné
• round_completed → Annoncer un round terminé

┌───────────────────────────────────────────────────────────────────┐
│ SERVEUR → CLIENT (Flutter reçoit)                               │
└───────────────────────────────────────────────────────────────────┘

• player_joined          → Un joueur a rejoint
• player_left            → Un joueur a quitté
• card_played            → Une carte a été jouée
• announcement_made      → Une annonce a été faite
• game_started           → Le jeu a démarré
• trick_won_broadcast    → Un pli a été gagné
• round_completed_broadcast → Un round est terminé
• error                  → Une erreur est survenue
• disconnect             → Déconnexion du serveur

💻 UTILISATION DANS FLUTTER
═══════════════════════════════════════════════════════════════════════

import 'package:cauris_app/services/websocket/game_websocket_service.dart';

// Créer une instance
final wsService = GameWebSocketService();

// Se connecter
await wsService.connect(serverUrl: 'ws://192.168.1.100:3000');

// Rejoindre une salle
await wsService.joinRoom('room123', 'Lewis');

// Écouter les événements
wsService.onCardPlayed().listen((data) {
  print('${data['playerName']} a joué une carte');
});

// Jouer une carte
await wsService.playCard(
  cardSuit: 'spades',
  cardValue: 'A',
  trickNumber: 1,
);

// Se déconnecter
await wsService.disconnect();
